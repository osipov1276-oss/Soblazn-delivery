SOBLAZN V8 MINI APP — ПОДКЛЮЧЕННАЯ ВЕРСИЯ

1. Распакуйте архив в отдельную папку.
2. Дважды нажмите start_soblazn_mini_app.bat.
3. При первом запуске вставьте настоящий токен от BotFather и нажмите Enter.
4. Токен сохранится в файле bot_config.env.
5. Откройте в браузере http://127.0.0.1:8080
6. Оформите тестовый заказ. Он должен прийти в группу с ID -1004342107012.

При следующих запусках токен вводить не нужно.
Для замены токена запустите reset_token.bat.

ВАЖНО:
— Не отправляйте файл bot_config.env другим людям.
— Не фотографируйте экран с токеном.
— Пока приложение работает локально, чёрное окно должно оставаться открытым.
