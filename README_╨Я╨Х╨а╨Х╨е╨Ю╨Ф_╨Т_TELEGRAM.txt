SOBLAZN V8 — ПРИВЯЗКА ЗАКАЗА К TELEGRAM

1. Запустите start_soblazn_mini_app.bat.
2. Откройте http://127.0.0.1:8080.
3. Оформите заказ.
4. На экране статуса нажмите «Открыть Telegram и получать статусы».
5. В Telegram нажмите кнопку START / ЗАПУСТИТЬ, если она появится.
6. Бот привяжет заказ к гостю и будет лично отправлять все изменения статуса.

Важно: одновременно не запускайте другой бот с тем же токеном, иначе появится Conflict.
