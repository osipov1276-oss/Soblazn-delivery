SOBLAZN V8 MINI APP — ПЕРВАЯ РАБОЧАЯ ВЕРСИЯ

1. Распакуйте папку.
2. Дважды нажмите start_local.bat.
3. Откройте в браузере: http://127.0.0.1:8080

Локальная версия нужна для проверки дизайна и корзины. Telegram Mini App принимает только публичный HTTPS-адрес, поэтому для подключения к боту папку нужно разместить на хостинге.

Для отправки тестовых заказов в группу перед запуском задайте BOT_TOKEN и COURIER_CHAT_ID в командной строке либо на хостинге. Не публикуйте токен внутри исходного кода.

После размещения на HTTPS: добавьте кнопку WebAppInfo(url="https://ВАШ-ДОМЕН") в бот и настройте Menu Button через BotFather.
