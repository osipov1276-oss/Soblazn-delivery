SOBLAZN V8 — статус заказа для гостя

1. Остановите старую версию Ctrl+C.
2. Распакуйте эту папку отдельно.
3. Запустите start_soblazn_mini_app.bat.
4. Откройте http://127.0.0.1:8080.
5. Оформите новый заказ.

После оформления откроется экран «Мой заказ».
Он автоматически обновляется каждые 4 секунды:
- Ожидает подтверждения
- Готовится
- Курьер принял
- Курьер в пути
- Доставлен

Если приложение открыто внутри Telegram, бот также отправляет гостю личные уведомления. Для этого гость должен сначала открыть бота и нажать Start.
